# timepy
Time stuffs

But seriously, print out the time in some timezones. 

## Requirements
- python 3.9+

## Install requirements
`pip3 install -r requirements.txt`

## Running
`python src/timepy.py`

## Installation
A possiblity:

`cp src/timepy.py <somewhere-in-your-path>/`
e.g. `cp src/timepy.py ~/bin/timepy`
